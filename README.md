## Installation

1. Clone the repository:

```bash
git clone https://github.com/Jovian-Dsouza/crypto-checkmate.git
```

2. Install dependencies
```
cd crypto-checkmate
npm install
```

3. Run development server
```
npm run dev
```

## Environment Variable
```
ABLY_API_KEY=... // Your ably API key
```
